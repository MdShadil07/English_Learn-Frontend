# Pricing components

This folder contains reusable pricing and subscription components used across:

- Landing / Pricing page
- Dashboard subscription management
- AI Chat subscription section
- Profile subscription section

Components:
- `PricingHero.tsx` — landing hero
- `PricingCard.tsx` — plan card
- `SubscriptionForm.tsx` — simple subscribe form (placeholder)
- `BillingTable.tsx` — compare features between plans

Styles are in `pricing.css`.
